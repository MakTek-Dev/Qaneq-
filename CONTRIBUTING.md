# Contributing to Qaneq

Qaneq is a community-driven Yup’ik language system focused on preserving, correcting, and expanding linguistic knowledge.

This project exists to:
- document accurate Yup’ik language usage
- identify and correct dictionary errors (including Jacobson)
- preserve cultural meaning and context
- allow knowledge to come from the people

---

## Ways to Contribute

### 1. Add a Translation
- Yup’ik → English
- English → Yup’ik
- Include dialect if known

### 2. Correct a Word or Meaning
- Identify incorrect translations
- Provide explanation and context
- Include regional usage if applicable

### 3. Audit Dictionary Entries (Jacobson Project)
- Flag errors or inconsistencies
- Provide corrected interpretation
- Include cultural or elder-based knowledge if possible

### 4. Add Cultural Context
- Explain how a word is actually used
- Include situations, tone, and meaning beyond literal translation

---

## Contribution Process

1. Fork the repository
2. Create a new branch
3. Make your changes
4. Submit a Pull Request

---

## Guidelines

- Be respectful of dialect differences
- Do not override another dialect — document differences instead
- Prioritize clarity, accuracy, and cultural respect
- When possible, include sources (elders, usage, experience)

---

## Mission

Qaneq belongs to the people.

This is not just a dictionary — it is a living system of language, culture, and identity.
