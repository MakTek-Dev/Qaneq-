# Qaneq

**Qaneq** is a Yup'ik-centered language system designed to translate, teach, and preserve **Kuskokwim Yup'ik**.

The word *Qaneq* refers to the **mouth**, and symbolically to the power of speech.

Words can hurt.  
Words can heal.  
That is the power of Qaneq.

This project aims to create tools that help people:

- translate between English and Yup'ik
- learn Yup'ik language structure and meanings
- understand cultural context behind words
- preserve Yup'ik knowledge for future generations

---

# Project Philosophy

Yup'ik language is not simply a collection of words.

It carries cultural knowledge, relationships with nature, and community values.

This system follows several principles:

- Respect the **All Yup'ik dialect**
- Avoid flattening Yup'ik into word-for-word English translation
- Teach the structure of the language, not just vocabulary
- Preserve cultural explanations and context
- Treat language as living knowledge

# Core Components

## Elitnaurista Qanyuuli

The AI teaching and translation engine.

Purpose:

- explain Yup'ik words and grammar
- analyze word roots and suffixes
- provide cultural explanations
- assist learners and translators

---

## Dictionary System

The system uses structured dictionary data.

Example location:

```
dictionary/
Yupik_dictionary.json
```

Each entry may contain:

- Yup'ik word
- English meaning
- root analysis
- suffix breakdown
- cultural notes
